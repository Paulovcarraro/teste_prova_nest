# teste_prova_nest

**Instalar NestJS CLI:**<br>

npm i -g @nestjs/cli

 **Criando o Projeto:** <br>
 
nest new biblioteca-api

Escolha: `Npm`

**Entrando na Pasta** <br>

cd biblioteca-api - `Cd + o nome do projeto`

**Rode para testar:** <br>

npm run start:dev

**Instalando MySQL** <br>

npm install @nestjs/typeorm typeorm mysql2

**se não der certo:** <br>

npm install @nestjs/typeorm@11 typeorm@0.3 mysql2

**Criar módulo de Livro**

nest g module livro
nest g controller livro
nest g service livro

`Mudar apenas o Nome do modulo que for usar`

**Instalar Biblioteca SWAGGER (interface)**<br>

npm install @nestjs/swagger swagger-ui-express
